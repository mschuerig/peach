Interesting findings:

Actually, in German, "Cent" is used for both singular and plural (established in Story 7.1 per Dev Notes). So a simple string is correct for German. This is fine.

2. **`%lld pitch matching exercises completed` German**: Lost plural variations. English has `"one": "%lld pitch matching exercise completed"`, `"other": "%lld pitch matching exercises completed"`. 

The German translation got flattened to `"%lld Tonhöhenübungen abgeschlossen"`, which would incorrectly show the plural form even for a single exercise. This is definitely a bug—German needs separate singular and plural forms: `"one": "1 Tonhöhenübung abgeschlossen"` and `"other": "%lld Tonhöhenübungen abgeschlossen"`.
