Some way to move a "magnifying lens" over time would be nice.

I'm not sure if it is sensible to form an aggregate over all training modes. At any rate, the Profile Screen should show a visualization for each training mode. Technical aside: this should not be implemented repeatedly. We should have a single, parameterized visualization and iterate over training modes.</command-args>
