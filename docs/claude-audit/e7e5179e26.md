## How to Update

The BMAD installer has a built-in update mechanism that preserves custom content. Run:

```bash
npx bmad-method install --action update
```

### Alternative: Quick Update

For a faster update that only refreshes changed files:

```bash
npx bmad-method install --action quick-update
```

## Safety Tips

1. **Commit first** — Make sure your working tree is clean (it is right now) so you can easily diff what changed
2. **Review the diff** after updating to see what was modified
