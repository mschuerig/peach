## Commit message

Implement story 34.1: CSV Import Parser

Add CSVImportParser (nonisolated enum) that reads the export CSV format
and converts rows back to ComparisonRecord/PitchMatchingRecord objects.
Includes RFC 4180 parsing, header validation, field-level validation with
error collection (no early abort), and interval abbreviation reverse lookup.

Also rename TuningSystem.storageIdentifier/fromStorageIdentifier to
identifier/init?(identifier:) — these are plain string parsers, not
storage-specific. The init is nonisolated since it's pure string matching.

72 new tests, 899 total passing.
