Create Story 3.3: Training Screen UI with Higher/Lower Buttons and Feedback

Story 3.3 adds visual and haptic feedback to complete the core training UX.

Key requirements:
- Add rounded rectangle corners (12pt radius) to Higher/Lower buttons (user request)
- Implement FeedbackIndicator component (thumbs up/down, green/red)
- Implement haptic feedback on incorrect answers (UIImpactFeedbackGenerator)
- No haptic on correct (silence = confirmation)
- Enable eyes-closed training via audio + haptic channels

Story context includes:
- Comprehensive UX analysis (sensory hierarchy: ears > fingers > eyes)
- Previous story intelligence (Story 3.2 already has feedback state)
- Architecture compliance requirements
- Implementation sequence and timing requirements
- User accessibility via haptic feedback

Status: ready-for-dev
