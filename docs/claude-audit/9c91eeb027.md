## Commit message

Implement story 8.2: SF2 preset discovery and instrument selection

Add SF2 PHDR binary parser, SoundFontLibrary service, and dynamic
instrument picker in Settings. Users can now choose from all pitched
SF2 presets (alphabetically sorted, unpitched/drums excluded).

Key changes:
- SF2PresetParser reads RIFF/sfbk PHDR metadata (287 presets from GeneralUser GS)
- SoundFontLibrary filters to pitched presets (bank<120, program<120), sorts A-Z
- SoundFontNotePlayer.loadPreset(program:bank:) switches presets with 20ms settle delay
- RoutingNotePlayer parses "sf2:{bank}:{program}" tags for multi-bank support
- Settings UI shows dynamic instrument list via @Environment(\.soundFontLibrary)
- Legacy "cello" tag migrated to "sf2:0:42"

Co-Authored-By: Claude Opus 4.6 <noreply@anthropic.com>
